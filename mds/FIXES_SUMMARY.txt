🔍 TROUBLESHOOTING

Still seeing sklearn warnings?
  ☐ Verify pandas installed: pip list | grep pandas
  ☐ Verify it's imported: python -c "import pandas"
  ☐ Check model files loaded: python -c "import pickle; pickle.load(open('pingpong_model.pkl'))"

Still no anchors after running everything?
  ☐ Check Flask is running (should see "Running on :8080")
  ☐ Check setup completed (should see "Applied to 11 UEs")
  ☐ Check detector is looking for ping-pongs (console should show activity)
  ☐ Lower P_pp threshold: --ppp-threshold 0.50
  ☐ Wait longer (first anchors may appear after 5-10 seconds)
  ☐ Check report: grep -A5 active_anchors ml_enhanced_report.json

Got "Cannot connect to http://localhost:8080"?
  ☐ Make sure app.py is running in Terminal 1
  ☐ Wait 2-3 seconds after starting app.py
  ☐ Check port 8080 is available: netstat -an | grep 8080
