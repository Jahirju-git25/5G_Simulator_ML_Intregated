  --visualize                 Enable RSRP plot generation [False]
  --viz-dir                   Plot output directory [handover_charts]
  --report                    Report output path [ml_enhanced_report.json]
  --verbose                   Print live decisions [False]

🔧 TROUBLESHOOTING QUICK FIX
═════════════════════════════════════════════════════════════════════════════════════

Problem: "Models not found"
Cause: pickle files missing or in wrong location
Fix: ls -la *.pkl  (must show both files)

Problem: "No module named 'matplotlib'"
Cause: Dependencies not installed
Fix: pip install matplotlib numpy scikit-learn requests

Problem: "Cannot connect to simulator"
Cause: Simulator not running or wrong URL
Fix: curl http://localhost:8080/api/get_state

Problem: "No HO events detected"
Cause: Thresholds too high
Fix: --ppp-threshold 0.50 --window 15.0

Problem: "Plots not generating"
Cause: --visualize flag not enabled
Fix: Add --visualize flag to command

Problem: "Invalid JSON in report"
Cause: Detector crashed or incomplete write
Fix: Check console for errors, run again


🎓 TECHNICAL DETAILS
═════════════════════════════════════════════════════════════════════════════════════

ML Model Architecture:
  Type: Sklearn LogisticRegression
  Inputs: 5 normalized features
  Output: P_pp ∈ [0, 1]
  Weights: Trained on simulator data
  Format: Pickle (.pkl)

Features Extracted:
  1. f_HO        - Handover frequency (HOs/sec)
  2. σ²_RSRP     - RSRP variance (signal stability)
  3. R_rev       - Reversal ratio (back-to-previous)
  4. D_flip      - Direction flips (A→B→A pattern)
  5. Osc         - Oscillation (3-step repeats)

Processing Pipeline:
  Raw Features → StandardScaler → LogisticRegression → P_pp
     ↓              ↓                   ↓              ↓
  Extract      Normalize        Trained Model    [0,1] Probability
               (mean=0,std=1)

Clustering Algorithm: DBSCAN
  - Epsilon (cluster_radius): 150 pixels
  - Min samples: 3 UEs per cluster
  - Metric: Euclidean distance

Anchor Optimization: Grid Search
  - Search radius: 60 pixels around centroid
  - Grid size: 5-pixel steps
  - Objective: Maximize minimum RSRP
